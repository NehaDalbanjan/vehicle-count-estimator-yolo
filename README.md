# vehicle-count-estimator-yolo
Developed a computer vision system to detect and count vehicles from CCTV footage using YOLOv8 and Python. Implemented object detection and tracking to classify vehicles and estimate traffic flow. The model processes video frames in real time and provides vehicle counts for traffic monitoring and analysis.

# Vehicle Count Estimator using CCTV Footage

This project detects and counts vehicles from traffic CCTV footage using Computer Vision.

## Technologies Used
- Python
- YOLOv8
- OpenCV
- NumPy

## Features
- Detect vehicles from traffic video
- Count vehicles automatically
- Works with CCTV footage
- Supports real-time detection

## Project Structure
vehicle_count.py – Main script for vehicle counting  
vehicle_detection.py – Vehicle detection logic  
live_detect.py – Live video detection  
video_test.py – Testing with sample video  

## How to Run
1. Install dependencies
2. Run the main file
